# 🤖 Cronuts AI Resources

Centro de recursos compartidos de inteligencia artificial para el equipo de Cronuts.

> Aquí encontrarás prompts, agents, skills y plantillas listas para usar. Si creas algo útil, ¡compártelo!

---

## 📁 Estructura del repositorio

| Carpeta | Contenido |
|--------|-----------|
| [`/prompts`](./prompts/) | Prompts organizados por área de trabajo |
| [`/agents`](./agents/) | Configuraciones de agentes AI completos |
| [`/skills`](./skills/) | Skills para Claude y otros modelos |
| [`/plantillas`](./plantillas/) | Plantillas reutilizables |
| [`/guias`](./guias/) | Guías de uso y buenas prácticas |

---

## 🚀 Cómo usar este repositorio

1. **Explorar**: Navega las carpetas y abre los archivos `.md` para ver el recurso.
2. **Copiar**: Copia el prompt o configuración directamente desde el archivo.
3. **Proponer mejoras**: Edita el archivo con el ícono ✏️ y abre una Pull Request.
4. **Contribuir**: Agrega tus propios recursos siguiendo la [plantilla estándar](./plantillas/plantilla-recurso.md).

¿Primera vez en GitHub? Lee la [guía de inicio rápido](./guias/como-usar-este-repo.md).

---

## 🏷️ Convención de nombres

Los archivos siguen el formato: `nombre-descriptivo-del-recurso.md`

Ejemplos: `prompt-copywriting-newsletter.md`, `agent-investigacion-competencia.md`
