# Cómo usar este repositorio (sin saber nada de GitHub)

Guía rápida para todo el equipo de Cronuts.

---

## Paso 1: Crear tu cuenta
1. Ve a [github.com](https://github.com) y haz clic en Sign up
2. Usa tu email de trabajo y un username como `nombre-cronuts`
3. Pídele al admin que te invite al repositorio

## Paso 2: Navegar
- El README de la página principal tiene el índice completo
- Haz clic en las carpetas para explorar
- Haz clic en un archivo .md para leerlo y copiar su contenido

## Paso 3: Editar un recurso existente
1. Abre el archivo → clic en ✏️ (lápiz)
2. Edita el texto
3. En "Commit changes": escribe un mensaje breve
4. Selecciona **"Create a new branch"** (nunca main directamente)
5. Clic en "Propose changes"

## Paso 4: Agregar un recurso nuevo
1. Entra a la carpeta correcta → "Add file" → "Create new file"
2. Nombre con extensión .md
3. Copia la plantilla de `/plantillas/plantilla-recurso.md` y rellénala
4. Mismo proceso de commit que arriba

## Paso 5: Pull Request
Después de proponer un cambio, abre una PR para que alguien lo revise antes de publicar.

---

## Glosario

| Término | Qué significa |
|---------|--------------|
| Repositorio | Carpeta principal con todos los archivos |
| Branch | Copia de trabajo para hacer cambios sin afectar main |
| Commit | Guardar un cambio con mensaje descriptivo |
| Pull Request | Propuesta de cambio que pasa por revisión |
| Main | La versión oficial publicada |
