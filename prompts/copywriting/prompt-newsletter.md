# Prompt: Copywriting para newsletter

> Genera el copy completo de una newsletter: asunto, preheader y cuerpo.

---

## Para qué sirve

Genera en un solo paso el asunto, preheader y cuerpo de una newsletter. Útil para primeros borradores rápidos.

---

## Cómo usarlo

1. Copia el prompt de abajo.
2. Reemplaza todo lo que está entre [CORCHETES].
3. Pégalo en Claude o ChatGPT.

---

## El recurso

```
Eres un copywriter experto en email marketing para la marca [NOMBRE DE MARCA].

El tono de la marca es: [DESCRIPCIÓN DEL TONO].
El público objetivo es: [DESCRIPCIÓN DE LA AUDIENCIA].

Escribe una newsletter completa sobre el tema: [TEMA O CAMPAÑA].
El objetivo es: [OBJETIVO — ej: anunciar producto, fidelizar, educar].

Incluye:
- Asunto (máximo 50 caracteres)
- Preheader (máximo 90 caracteres)
- Cuerpo: apertura impactante, desarrollo, llamada a la acción clara

No uses frases genéricas como "esperamos que estés bien".
```

---

## Ejemplo de uso

**Input:** Marca: Cronuts Studio | Tono: cercano y directo | Tema: nuevo servicio de auditoría paid media

**Output esperado:** Asunto + preheader + email completo listo para editar.
